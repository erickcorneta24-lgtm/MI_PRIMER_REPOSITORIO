# MI_PRIMER_REPOSITORIO

Este repositorio fue creado como parte de una práctica para aprender los conceptos básicos de GitHub.  
Aquí organizo una estructura de carpetas con archivos CSV simples para practicar la creación, edición y subida de archivos al repositorio

## Estructura del proyecto

mi-primer-repositorio/
│
├── datos_ventas/
│ └── ventas.csv
│
└── datos_clientes/
└── clientes.csv

## Archivos incluidos

### datos_ventas/ventas.csv
Contiene información de ventas de ejemplo (ID, fecha, producto, cantidad y total).

### datos_clientes/clientes.csv
Incluye datos ficticios de clientes (ID, nombre, correo y ciudad).

##  Objetivo del repositorio
- Aprender a crear carpetas y archivos desde GitHub.  
- Practicar la subida de contenido (CSV).  
- Familiarizarme con la estructura de un proyecto.  
- Explorar lo que sería mi primer repositorio formal en GitHub.

##  Mi experiencia
Este es mi primer repositorio en GitHub y me sirve para entender cómo funciona el control de versiones, cómo organizar archivos y cómo documentar un proyecto básico.
